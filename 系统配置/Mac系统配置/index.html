2. Homebrew安装

  • Homebrew是什么？用Homebrew官网上的一句话介绍；

    “使用 Homebrew 安装 Apple 没有预装但 你需要的东西。”

  • 也就是说，Homebrew是一个第三方的Mac软件安装、管理工具（第三方应用商店）；

  • Homebrew的安装需要在Mac终端（命令行）进行，直接在本机终端内，复制黏贴官网给出的命令然后回车执行即可，官网给出的命令如下
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"


3. 安装iterm2+oh-my-zsh

  • 使用Homebrew cask（cask用来安装Mac下的图形化软件）安装iterm2
brew cask install iterm2
  • 样式调整：打开iterm2后，按快捷键command, 打开属性菜单，选择Profiles - 通过Colors（调整主题配色）| Text（调整字体）选项卡进行相应设置，需要注意左侧的Profiles Name栏，调整都要针对当前默认配置进行（有星标的），否则，进行的调整不会有任何变化
  • 安装oh-my-zsh
sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
  • 通过cat /etc/shells命令可以查看当前系统可以使用哪些shell
  • 切换系统默认shell，如果当前的shell不是zsh，我们可以通过chsh -s /bin/zsh命令将shell切换为zsh，终端重启之后即可生效
  • 安装成功之后，我们可以通过终端命令vi ~/.zshrc打开隐藏的配置文件，编辑设置ZSH_THEME="agnoster"对主题进行修改

  • 注意，agnoster主题能否设置成功，还依赖于以下东西：

    • 下载完成之后解压，在iTerm2的Preferences——Profiles——colors——Load Presets——Solarized Dark即可设置终端配色 http://ethanschoonover.com/solarized
    • 下载、安装Powerline 对应的字体库 bash git clone https://github.com/powerline/fonts.git
    • 下载后，进入目录，执行其中的install.sh文件，命令：./install.sh，然后在iTerm2的Preferences——Profiles——Text中同时将Regular Font和Non—ASCII Font设置为Meslo LG M DZ Regular for Powerline即可


4. VScode安装和配置

  • 在微软官网下载VScode，然后安装就好了
  • 安装完毕后打开VScode，VScode需要安装各种插件才能发挥神一般的能力。在欢迎页面的自定义栏目，可以对语言、快捷键和主题做设置，希望设置什么就点击相应的栏目就可以了

  • VScode的插件默认是按照下载量降序显示的，常规插件就不提了，这里重点推荐一款Markdown插件，简直神了，支持实时预览，通过第三方插件可以导出、导入多种类型的文件，关键是这个插件有中文文档，学习如何使用简直不要太简单

    • 插件名称：Markdown Preview Enhanced
